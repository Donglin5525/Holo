# HoloAI 记忆闭环与显性感知重构 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use executing-plans to implement this plan task-by-task.

**Goal:** 删除 HoloAI 旧格式记忆及其全部兼容入口，只保留可验证的新语义记忆，并让普通对话、洞察和 Agent 能按问题相关性召回且向用户展示真实使用回执。

**Architecture:** 使用严格 V2 领域模型与一次性宽松迁移 DTO 隔离旧 JSON；所有候选通过 `subjectKey` 生成稳定身份并合并证据；`HoloMemorySnapshotBuilder` 统一召回，Chat 用尾标记确认实际引用，Agent 通过 Tool/Evidence Ledger 预取。Memory Curator 本轮保持只读路由器，不写 Store。

**Tech Stack:** Swift 6、SwiftUI、Codable、UserDefaults、Core Data、standalone `swiftc` tests、Xcode simulator build、Hono、Node.js `node:test`。
