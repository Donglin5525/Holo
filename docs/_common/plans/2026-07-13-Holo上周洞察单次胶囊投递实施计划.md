# Holo 上周洞察单次胶囊投递 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use executing-plans to implement this plan task-by-task. 当前项目明确禁止 subagent，本会话由主 agent 逐项执行并在每项后验证。

**Goal:** 将周洞察统一为上一完整自然周的自动产物，通过首页 P0 胶囊单次投递，并精准打开记忆长廊对应洞察，同时修复空响应、超时和失败覆盖成功内容的问题。

**Architecture:** 继续使用 `MemoryInsight(.weekly)` 保存成功资产；新增纯逻辑的上一周周期与胶囊投递判定，Repository 严格区分可用洞察和生成尝试。Deep Link 携带 insight UUID 驱动记忆长廊切换到洞察 Tab 并展开 AI 回放；后端对洞察空正文和截断做一次自动重试与稳定错误映射。

**Tech Stack:** Swift 6、SwiftUI、Core Data、UserDefaults、Hono、Node.js、node:test、standalone `swiftc` tests、Xcode simulator build。
