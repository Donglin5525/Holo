# 财务模块自定义分类管理功能

## 背景

用户希望在财务模块设置中管理自定义分类，包括：
- 查看已创建的分类列表
- 新增自定义分类（选择一级分类 → 输入名称 → 选择图标）

## 现有架构

```
FinanceView
├── FinanceTabView (统计)
├── LedgerListView (账本)
└── FinanceSettingsView (设置)
    └── ImportExportView (导入导出)  ← 目前只有这个

CategoryManagementView  ← 已存在，入口在 CategoryPicker
```

---

## 实现方案

### 1. 在财务设置中添加"分类"入口

**修改文件：** `Views/FinanceView.swift` 中的 `FinanceSettingsView`

```swift
// FinanceSettingsView 新增
Section("数据管理") {
    // 现有的导入导出
    ImportExportView()
}

Section("分类管理") {
    NavigationLink("分类") {
        CategoryManagementView()  // 复用现有组件
    }
}
```

### 2. 新增分类流程（在 CategoryManagementView 的 AddCategorySheet）

**用户流程：**
```
1. 选择一级分类（支出/收入的顶级分类）
   ↓
2. 输入二级分类名称
   ↓
3. 从 82 个预设图标中选择
   ↓
4. 保存
```

**AddCategorySheet 改造：**

```
┌─────────────────────────────────┐
│  新增分类                     [×]│
├─────────────────────────────────┤
│  一级分类                        │
│  ┌─────────────────────────────┐│
│  │ 餐饮                    ▼  ││  ← 选择器
│  └─────────────────────────────┘│
├─────────────────────────────────┤
│  二级分类名称                    │
│  ┌─────────────────────────────┐│
│  │ 请输入名称...              ││  ← 文本输入
│  └─────────────────────────────┘│
├─────────────────────────────────┤
│  图标                            │
│  ┌───┐ ┌───┐ ┌───┐ ┌───┐       │
│  │🍽️│ │🚗│ │🛒│ │🎮│ ...     │  ← 图标网格
│  └───┘ └───┘ └───┘ └───┘       │
├─────────────────────────────────┤
│           [取消]  [保存]         │

```

### 3. 创建图标选择组件

**新文件：** `Components/IconPickerGrid.swift`

- 展示 82 个预设图标
- 4 列网格布局
- 单选模式
- 选中高亮

## 实现步骤

### Step 1：添加设置入口
- 修改 `FinanceSettingsView`，添加"分类管理" Section
- 点击跳转到 `CategoryManagementView`

### Step 2：改造 AddCategorySheet
- 将文本输入改为一级分类选择器
- 保留名称输入
- 将图标输入改为图标网格选择

### Step 3：创建 IconPickerGrid 组件
- 硬编码 82 个图标名称列表
- 4 列 LazyVGrid 布局
- Binding 传递选中图标

### Step 4：预设图标数据
- 从 Asset Catalog 提取 82 个图标名称
- 定义为常量数组
